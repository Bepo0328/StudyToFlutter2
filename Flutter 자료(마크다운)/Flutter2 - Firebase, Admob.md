# 0. Firebase, Admob
	1. Firebase

***
# 1. Firebase
	- Firebase
	Firebase란 모바일 앱을 분석, 테스트 뿐만 아니라 개발, 서비스까지 할 수
    있는 모바일 개발 플랫폼
      - Analytics: 사용자 분석
      - Firestore: 데이터 저장 및 불러오기
      - Crashlytics: 모바일 오류 분석
      - Authentication: 사용자 인증

***
# 2. Admob
	- Admob
	Admob란 구글에서 제공하는 모바일 광고 플랫폼
    앱을 통해서 수익을 창출 할 수 있는 방법
      - 배너 광고 - 화면 하단에 노출
      - 전면 광고 - 화면 전체 노출
      - 보상형 광고 - 게임 아이템 얻기
      
      admob_flutter 라이브러리 사용 예정
